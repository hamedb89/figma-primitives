The sharp edges of Figma plugins, handled.

A small set of typed primitives for messaging, async tasks, selections, networking, and plugin UI. Product decisions stay yours.

Get started →
tasks.tsTypeScript
import { createTaskEngine } from 'figma-primitives'

const tasks = createTaskEngine()

await tasks.run(
  { type: 'selection.inspect', label: 'Inspect selection' },
  async ({ signal, progress }) => {
    progress({ label: 'Reading layers' })
    return inspectSelection({ signal })
  },
)
